--조건부서식
<관계연산자>
>=  이상 예) A1>=10
>  초과   예) A1>10
<= 이하  예) A1<=10
<  미만   예) A1<10
=  같다   예) A1=10
<>  같지않다. 예) A1 <> 10

<논리연산자>
AND ~이면서, ~이고(모든 조건이 만족)
OR ~이거나,~또는(조건중에서 하나 이상만 만족)

와일드 카드
* ->모든문자, ?-> 하나의 문자
*일반 --> 일반으로 끝난다.
일반* --> 일반으로 시작한다.
*일반* --> 일반을 포함한다.
일반?? --> 일반으로 시작하고 총4글자
?일반 --> 일반으로 끝나는 3글자
